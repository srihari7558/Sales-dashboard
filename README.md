# 📊 Sales Data Analysis Dashboard

## 📌 Project Overview

This project analyzes sales data and presents insights using an interactive Streamlit dashboard.

## 🚀 Features

* KPI metrics (Total Sales, Orders, Average Sales)
* Region & Category filters
* Monthly sales trend
* Top performing products
* Interactive visualizations

## 🛠 Tech Stack

* Python
* Pandas
* Streamlit

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
